## Description

A full stack web application with infrastructure to support image/video upload/download to AWS. 

Frontend Tech:
- React
- Vite
- Shadcn

Backend Tech:
- AWS ApiGateway
- AWS Lambda
- AWS S3
- AWS Cognito
